# The following is a list of all of the commands available in startpage terminal
# Click the back button or press alt + left arrow to go back to the startpage

## NOTE ## Your custom commands from json/data.json will not show up in here! ## NOTE ##

## GENERAL ##
[URL]
    Goes straight to that url, as long as it is valid.
[search query]
    Searches the query
help | commands | cmd | ?
    Show this list

## SEARCH ENGINE ##
set [arg]
    Sets search engine to given arg.
        - The textbox will clear if it SUCCEED.
        - The textbox will not clear if it FAILED.
        
    Available search engines can be found in json/data.json with "searchURL".
